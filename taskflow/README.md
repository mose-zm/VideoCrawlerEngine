VideoCrawlerEngine
===============

# 视频爬取引擎

视频爬虫引擎是以自定义脚本为草图，生成请求器工作流，实现可控可视的流水线执行。
意在提高脚本的开发效率，让开发者专注于解决反爬。

## WEB UI

![UI 1](../docs/UI_1.png)
![UI 2](../docs/UI_2.png)

## 环境搭建
> pip install -r requirements.txt

## 使用方法
> python main.py

- 开启本地服务器(默认端口2333)

- 浏览器打开 http://localhost:2333

## Requirements

- Python >= 3.6
- Nbdler >= 3.0.3
- PyJSCaller >= 0.2.0
- Flask
